# Security policy

If you believe you've found a security issue in Stowage, please report
it privately first. **Do not open a public GitHub issue.**

The preferred channel is **GitHub Security Advisories**:

1. Open <https://github.com/stowage-dev/stowage>.
2. Go to **Security → Advisories → Report a vulnerability**.
3. Submit your report.

This routes the report to the maintainers privately and gives us a
shared workspace to triage and patch.

For full details — what to include in a report, scope, response SLAs,
and safe-harbour terms — see
[`docs/security/report-vulnerability.md`](./docs/security/report-vulnerability.md).
